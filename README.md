# E-Commerce Sales Dashboard (Power BI)

## Overview
This project is an interactive Power BI dashboard designed to analyze e-commerce business performance. It provides insights into sales, customers, products, reviews, and orders through interactive visualizations and slicers.

## Dashboard Features
- KPI Cards
  - Total Users
  - Total Orders
  - Total Products
  - Total Sold
  - Average Rating
  - Total Reviews

- Sales Analysis
  - Monthly User Registrations
  - Top Selling Products
  - Orders by Category

- Customer Analysis
  - Users by Brand
  - Product Reviews
  - Customer Insights

- Interactive Filters
  - Category
  - Brand
  - Date
  - Rating

## Tools Used
- Power BI Desktop
- DAX
- Data Modeling
- Power Query
- Excel/CSV Dataset

## Skills Demonstrated
- Data Cleaning
- Data Modeling
- DAX Measures
- Dashboard Design
- Interactive Visualizations
- KPI Development
- Business Intelligence

## Project Outcome
The dashboard enables users to monitor sales performance, identify top-selling products, analyze customer behavior, and make data-driven business decisions.
